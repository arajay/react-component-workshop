# List Component Requirements
- should render a list
- should include a control to switch between list types (bullet, number, icons)
- should be able support icons as bullets
- should support hyperlinks in the text
- should support nested lists
- should support different text styling for list content
- should support different colors for bullet and text
- should have an unstyled option
